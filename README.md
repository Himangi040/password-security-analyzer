# Password Security Analyzer

A Python-based tool that evaluates password strength using entropy calculations and demonstrates secure password hashing.

## Features
- Password strength evaluation
- Entropy-based scoring
- Secure hashing using SHA-256
- Output logging to file

## Tools & Technologies
- Python
- Cryptography fundamentals
- Secure password handling

## Disclaimer
This project is for educational purposes only and demonstrates best practices for password security.
